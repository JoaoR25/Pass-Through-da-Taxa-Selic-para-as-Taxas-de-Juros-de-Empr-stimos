# Pass-through da Taxa Selic para as Taxas de Juros de Empréstimos no Brasil

Este repositório contém os scripts e a base de dados utilizados no Trabalho de Conclusão de Curso de João Gabriel Rodrigues da Rosa, intitulado *O pass-through da Taxa Selic para as taxas de juros de empréstimos no Brasil*.

## Arquivos do repositório

- `Dados trabalho.xlsx`: base de dados utilizada na pesquisa.
- `ADF e PP.R`: script com os testes de estacionariedade ADF e Phillips-Perron.
- `Teste Johansen.R`: script utilizado para o teste de cointegração de Johansen.
- `Modelo_PASS-THROUGH DA TAXA SELIC.R`: script principal relacionado ao modelo do trabalho.

## Observação

Os scripts foram desenvolvidos em linguagem R e organizados para fins acadêmicos, com o objetivo de auxiliar na análise empírica do Trabalho de Conclusão de Curso.
